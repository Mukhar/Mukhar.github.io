// About

Engineer first, problem-solver always.

I like the hard, unglamorous middle of the stack — the services that don't ship with a UI but make everything else possible.

I graduated from IIT Delhi in 2021 and have spent the last five years building systems that quietly carry a lot of weight — onboarding platforms used by thousands, secrets managers that touch every microservice, and data pipelines that feed the dashboards leadership reads every morning.

At Walmart Labs I work full-stack — Java and Spring Boot for the heavy lifting, Go for systems work, React for the UI, Kubernetes and GCP underneath. Lately I've been focused on bringing GenAI into developer workflows, where the gains compound.

Before Walmart I spent three years at Standard Chartered leading a Java 8 → 17 modernization, rebuilding the bank's digital onboarding, and saving $1M/year by redesigning a single PDF utility.
